Selenide

Selenide is a library for writing concise, readable, boilerplate-free tests in Java using Selenium WebDriver.

Concise UI Tests in Java

@Test
public void userCanLoginByUsername() {
  open("/login");
  $(By.name("user.name")).setValue("johny");
  $("#submit").click();
  $(".loading_progress").should(disappear); // Waits until element disappears
  $("#username").shouldHave(text("Hello, Johny!")); // Waits until element gets text
}

Selenide is a wrapper for Selenium WebDriver providing:

  • Automatic handling of browser lifecycle
  • jQuery-like element selectors
  • Ajax support

That way Selenide provides a Concise API for writing tests. You don't need to think how to shutdown browser - just concentrate on your business logic!

How to start?

Just put selenide.jar to your project. For Maven, Ivy, Gradle etc., users: 
<dependency org="com.codeborne" name="selenide" revision="2.1"/>
See Quick Start guide for more details
